# Jonatan Zuniga Portfolio

Portfolio personal construido con Next.js, React, TypeScript y Tailwind CSS.

## Estructura

```txt
app/          Rutas principales de Next.js
components/   Secciones visuales reutilizables
data/         Contenido editable del portfolio
lib/          Tipos y utilidades compartidas
public/       Imagenes, assets y CV
```

## Scripts

```bash
npm run dev
npm run build
npm run start
```

## Edicion de contenido

La informacion principal del sitio vive en `data/portfolio.ts`: navegacion, perfil,
experiencia, proyectos, certificaciones y enlaces de contacto.

Las imagenes publicas viven en `public/assets/` y el CV en `public/cv/`.
